# miniguia-estudos-notebooklm
## Contexto

Esse MiniGuia foi criado com o intuito de enfatizar e aprofundar os meus estudos em redes de computadores e da base da tecnologia, onde trata-se de um tema incrível que muitas vezes profissionais da área não entendem direito e em que o público em geral não dá valor de saber, que é "Como a internet funciona ?". Neste miniguia,juntei algumas fontes interessante, pedi para o NotebookLM focar nelas e consigo trazer algumas respostas que acabei gostando além do guia. 

## Fontes

1. Mozilla Developers - https://developer.mozilla.org/pt-BR/docs/Learn_web_development/Howto/Web_mechanics/How_does_the_Internet_work
2. Escola Linux - https://nova.escolalinux.com.br/blog/como-funciona-a-internet-entenda-de-uma-vez-por-todas
3. Cloudfare - https://www.cloudflare.com/pt-br/learning/network-layer/how-does-the-internet-work/
4. Tecnoblog - https://tecnoblog.net/responde/o-que-e-internet-entenda-o-funcionamento-da-rede-global-de-computadores/
